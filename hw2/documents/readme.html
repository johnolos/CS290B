Compile

ant compile

Jar

How to run

Space:*

ant -Dserver-ip=0.0.0.0 runSpace

Computer:

ant -Dserver-ip=0.0.0.0 runComputer

Clients:

ant -Dserver_ip=0.0.0.0 runMandelbrot

ant -Dserver_ip=0.0.0.0 runTSP

Javadoc

ant javadoc

Clean

ant clean

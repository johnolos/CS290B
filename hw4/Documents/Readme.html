Clean, Compile, Jar

ant all

This coman run ant clean compile jar

How to run

Space:

ant runSpace or..

ant -Dserver-ip=x.x.x.x runSpace *

Computer:

ant runComputer or..

ant -Dserver-ip=x.x.x.x runComputer *

ant -Dmulticore="true"/"false" runComputer ***

ant -DamelioriteLatency="true"/"false" runComputer ****

Clients:

ant runFib or..

ant -Dnum=x -Dserver-ip=x.x.x.x runFib **

ant runTSP

* The IP is set to default (localhost), but it is possible to override it with your own choice of IP by attaching the -Dserver-ip command.

** The fibonacci number is set to default (20), but it is possible to overrised it with your own choice of number by attaching the -Dnum=x command. x is representing the fibonacci number.

*** -Dmulticore is to set the value either true or false. The default setting for this is true.

**** -DamelioriteLatency is to set the value either true or false. The default setting is true.

Initaite a Space, computer(s) and then run task(s). Make sure a task is completed before running another one.

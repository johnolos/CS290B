Clean, Compile, Jar

ant all

This command run ant clean compile jar

How to run

Space:

ant runSpace or..

ant -Dserver-ip=x.x.x.x runSpace *

Computer:

ant runComputer or..

ant -Dserver-ip=x.x.x.x runComputer *

Clients:

ant runFib or..

ant -Dnum=x -Dserver-ip=x.x.x.x runFib **

ant runTSP

Javadoc

ant javadoc

* The IP is set to default (localhost), but it is possible to override it with your own choice of IP by attaching the -Dserver-ip command.

** The fibonacci number is set to default, but it is possible to overrised it with your own choice of number by attaching the -Dnum=x command. x is representing the fibonacci number.
